HTML Table Tag:

	<td></td> td na table date, td represent panurathu table oda column ah tha.

<table border="1">
    <tr>
        <td>Name</td><td>Age</td><td>City</td>
    </tr>

    </tr>
    <tr>
        <td>Tom</td><td>20</td><td>Mumbai</td>
    </tr>
</table>

tr - Table Row
td - Table Column or Table Data
border="1" - Gives you border around table
