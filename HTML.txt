Image Tag:
	<img src="image location" width="100px" alt="alternative text if the image doesn't load"/>
attribute "src" means "Source", "alt" vanthu "Alternative text" ipo nama thedura image kedikalana in the "alt" kula ena eluthi irukomo athu work agum, "width" attribute vanthu image size ah set pana use agum.
Ex: <img src="vijayimage.jpeg" alt="Vijay Image" with="100px">

	</ul>
